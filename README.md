## 📖 Overview

ShopLane is a responsive front-end e-commerce website that delivers a smooth online shopping experience. It includes dynamic product browsing, product details, search functionality, shopping cart management, and a clean user interface designed to provide an intuitive shopping journey.

---

## ✨ Features

- 🏠 Responsive Home Page
- 🔍 Product Search
- 🛒 Shopping Cart
- 📦 Product Details Page
- 👕 Clothing & Accessories Categories
- 💰 Price Display
- 📱 Mobile-Friendly Design
- 💾 LocalStorage Cart Management
- ✅ Order Confirmation Page
- 🎨 Modern & Clean UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage API
